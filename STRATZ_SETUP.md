# STRATZ API підключення

У проєкті вже додано готове підключення STRATZ API.

## Як користуватись

1. Запусти програму:

```powershell
python main.py
```

2. Відкрий сторінку **Налаштування**.
3. У полі **STRATZ API ключ** встав свій токен.
4. Натисни **Перевірити**. Якщо ключ правильний, він одразу збережеться в локальну SQLite-базу.
5. Перейди на сторінку **Статистика героїв**.
6. Натисни **Оновити статистику**.

Після оновлення програма бере:

- **Win Rate / Pick Rate** зі STRATZ API;
- **атрибут, роль, тип атаки, складність** з OpenDota metadata;
- **опис і Ban Rate** з локальної бази, якщо вони є.

API-ключ не записується в код і не потрапляє в GitHub. Він зберігається тільки локально в SQLite.
