# Gemini AI setup

Ця версія програми має підключення Gemini API.

## Що працює

1. Якщо комп’ютерний зір розпізнав не повну команду, наприклад 3 союзники замість 5, програма автоматично додає рекомендованих героїв у вашу команду.
2. Якщо Gemini API ключ є — автодоповнення робить Gemini.
3. Якщо ключа немає або API не відповів — програма використовує локальний алгоритм рекомендацій, щоб не вилітати.
4. Блок «Пояснення від ШІ» генерує текстове пояснення рекомендацій.

## Як підключити ключ

1. Відкрий програму.
2. Перейди в «Налаштування».
3. У полі «API Key Gemini» встав ключ.
4. Натисни «Перевірити».
5. Якщо перевірка успішна, натисни «Зберегти налаштування».
6. Перейди на головну сторінку, завантаж скриншот і натисни «Почати аналіз».

## Де код

- `services/gemini_service.py` — підключення Gemini API, AI-пояснення, AI-доповнення драфту.
- `ui/page_home.py` — логіка: якщо союзників менше 5, додати рекомендованих героїв.
- `ui/page_settings.py` — перевірка Gemini API ключа.

## Як змінити модель

У файлі `services/gemini_service.py` знайди:

```python
GEMINI_MODEL = "gemini-3.5-flash"
```

і заміни на модель, до якої має доступ твій ключ.
